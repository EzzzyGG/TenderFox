# Стратегический план развития (12 месяцев)

### Q1: MVP и Первые продажи
- Запуск парсинга 3 основных площадок.
- Реализация скоринга "Надежность оплаты".
- Привлечение первых 50 платных подписчиков через таргет.

### Q2: Масштабирование
- Добавление коммерческих площадок (B2B-Center, Фабрикант).
- Модуль автоматической проверки банковских гарантий.
- Мобильное приложение (PWA).

### Q3: ИИ-Ассистент
- Внедрение LLM для анализа документации (поиск "ловушек" в ТЗ за секунды).
- Автогенерация запросов на разъяснение документации.

### Q4: Экосистема
- Маркетплейс тендерных специалистов.
- Интеграция с сервисами финансового учета.
