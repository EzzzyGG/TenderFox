# Стратегия Performance-маркетинга для TenderWin

## Целевая Аудитория (РФ)
1. Владельцы малого бизнеса (строительство, поставки, клининг).
2. Индивидуальные предприниматели (услуги).
3. Тендерные специалисты на фрилансе.

## Рекламные Каналы
### Яндекс.Директ (Поиск + РСЯ)
- Ключи: "сервис для тендеров", "аналитика госзакупок", "проверка заказчика по инн".
- Оффер: "Не трать время на тендеры с задержкой оплаты. Бесплатный скоринг заказчика внутри."

### VK Ads (Таргет на предпринимателей)
- Парсинг сообществ: "Бизнес Молодость", "Мой Бизнес", группы про госзакупки.
- Креатив: "Узнай свои шансы на победу в тендере за 30 секунд. Ссылка в описании."

## Воронка (Freemium)
- Бесплатно: поиск по всем тендерам + 3 полных отчета по заказчикам.
- Платно: безлимитный доступ + уведомления в Telegram + ИИ-анализ рисков.
