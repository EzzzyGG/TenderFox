TenderFox

Поиск тендеров → фильтры → подписка → уведомления в Telegram.

Открыть бота API Docs

Локально: API http://localhost:8000, сайт http://localhost:8080

Что делает

  • Ищет тендеры по ключевым словам и региональным фильтрам
  • Позволяет подписаться на фильтр
  • Отправляет уведомления в Telegram

Как начать

  1. Скопируй .env.example → .env
  2. Заполни TELEGRAM_BOT_TOKEN
  3. Запусти docker compose up --build
