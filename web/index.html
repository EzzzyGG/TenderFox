Тендеры, которые тебе нужны — прямо в Telegram

Пишешь запрос (например, «мебель 77») — TenderFox находит новые тендеры и присылает уведомления.

Как работает

  1. 1. Открываешь бота
    Нажимаешь «Запустить» в Telegram.
  2. 2. Добавляешь подписку
    Команда: /add мебель 77 (ключевые слова + код региона).
  3. 3. Получаешь новые тендеры
    TenderFox периодически проверяет новые закупки и отправляет только новые (без дублей).

FAQ

Это бесплатно?
Сейчас — да (MVP). Дальше возможен freemium.
Какие регионы?
Указывай код региона РФ (например, 77). В интерфейсе позже добавим выбор из списка.

Потому что это самый быстрый канал: подписка → уведомления без личного кабинета.
