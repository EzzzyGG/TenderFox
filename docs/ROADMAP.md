# TenderWin PRO: Агрегатор и Аналитик Тендеров для РФ

Система поиска тендеров со встроенным скорингом надежности заказчиков.

## КИЛЛЕР-ФИЧА: Corruption & Risk Score
Алгоритм анализирует ИНН организатора по открытым реестрам и выдает:
1. Вероятность задержки оплаты (на основе арбитражных дел).
2. Риск "заточенного" тендера (если один и тот же ИНН выигрывает у этого заказчика 80%+ случаев).
3. Вероятность победы для новичка.

## ТЕХНОЛОГИЧЕСКИЙ СТЕК
- Backend: Python 3.10 + FastAPI
- База: PostgreSQL (через SQLAlchemy)
- Очереди: Redis + Celery (для фонового парсинга)
- Платежи: ЮKassa (API v3)
- Тесты: Pytest

## КАК ЗАПУСТИТЬ
1. Установите Docker и Docker Compose.
2. Скопируйте файл `.env.example` в `.env` и вставьте свои ключи.
3. Выполните: `docker-compose up --build -d`.
4. API будет доступно по адресу: `http://localhost:8000/docs`.
